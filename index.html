Schedule Optimizer

Aug 2017 - Dec 2017

Schedule Optimizer uses computation to generate customized class schedules without users having to deal with details. Users say which classes they want to take, our website figures out how to make it work. I was responsible for developing an intuitive frontend using React and interacting the our team's backend API. This project initially started as a project for UIUC's CS 196: Freshman Honors course.
