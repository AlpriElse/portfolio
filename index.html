Schedule Optimizer

Started Fall 2017

An evolution from Course Explorer, Schedule Optimizer uses computation to generate customized class schedules without dealing with the details. Users say which classes they want to take, our website figures out how to make it work. We later plan to make more user personalized features such as professor preference, four year plans, and more.This project initially started as a project for UIUC's CS 196: Freshman Honors course.

ReactJS Frontend

Responsible for modeling course data in ReactJS for seamless user experience. Interfaced with backend API to display course data and dynamically generated schedules.

Team Project

Followed team conventions and git workflow in order to effectively collaborate with team members. Used Github stored codebase to stay syncronized with team members.

MLH: Hack Ridge 2017

Fall 2016 - Spring 2017

Organizer for Major League Hacking sponsored high school hackathon, Hack Ridge. Hack Ridge 2017 was a 24-hour open hackathon for high schoolers in the area and accomodated over 100 student hackers, many coming from other high schools. Sponsors include web design company American Eagle and programming course service Code School. Contributed to all aspects of the organizing process, focusing on marketing efforts directed toward Maine South H.S. students.
